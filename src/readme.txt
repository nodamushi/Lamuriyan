-*Japanese*-
**srcディレクトリについて**
srcディレクトリ直下のディレクトリは、Eclipseプロジェクト内でソースフォルダーになります。

src
├converter :　内部構造をHTML（DOM）に変換する
├core      :　Lamuriyanのパーサー関連
├core_io   :　ファイル読み込み関連
├core_macro:　マクロ関連。
├core_node :　DOMライクなデータ構造関連
├test      :　なんかごちゃごちゃするためのディレクトリ。Mainもここになぜか。
└readme.txt:　このファイル

↓間違いを誰か直してください


-*English*-
**about src directory**
The directories under src dirctory are source folder in Eclipse propject.

src
├converter : convert lamurina datas to HTML DOM.
├core      : a parser of Lamuriyan.
├core_io   : IO.
├core_macro: native macros.
├core_node : the inner data structure likes DOM.
├test      : not for JUnit.Somehow test directory has Main.
└readme.txt: this file.
